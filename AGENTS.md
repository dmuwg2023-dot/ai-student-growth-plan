# Repository Agent Instructions

This repository contains a universal AI Student Growth Plan workflow.

When an AI coding or document agent works in this repository:

- Treat `UNIVERSAL_AGENT.md` as the platform-neutral entrypoint.
- Treat `ai-student-growth-plan/` as the Codex skill adapter.
- Keep the workflow focused on college student growth planning and supervision:
  `deep self-analysis -> five mentor agents -> chief planner -> daily accountability -> weekly review`.
- Do not add video-production, marketing-site, or unrelated content unless explicitly requested.
- Keep prompts reusable across AI platforms.
- Keep safety boundaries: do not diagnose mental health conditions, replace professional counseling, guarantee outcomes, or encourage unsafe overwork.

Main files:

- `UNIVERSAL_AGENT.md`: copy-paste universal agent instructions.
- `INSTALL.md`: platform-specific usage guide.
- `ai-student-growth-plan/SKILL.md`: Codex skill entry.
- `ai-student-growth-plan/references/agents.md`: agent definitions.
- `ai-student-growth-plan/references/prompt-library.md`: reusable prompts.
- `ai-student-growth-plan/references/output-formats.md`: output tables.
